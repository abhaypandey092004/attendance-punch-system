# 🏢 Attendance Punch System

A simple Attendance Management System built using Python, Tkinter, SQLite, and OpenPyXL.

---

## 🚀 Features

- ✅ User Login & Registration
- ✅ Admin & Employee Dashboard
- ✅ Punch In / Punch Out System
- ✅ Automatic Working Hours Calculation
- ✅ Leave Management System
- ✅ Export Attendance to Excel
- ✅ Export Leave Records to Excel

---

## 🛠 Technologies Used

- Python 3
- Tkinter (GUI)
- SQLite (Database)
- OpenPyXL (Excel Export)
